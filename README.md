# Global Superstore SQL + Tableau Project

This project explores the [Global Superstore dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final) using **SQL (BigQuery)** and **Tableau Public**.

## Questions Answered
1. What are the top 10 most profitable products?
2. Which countries/regions generate the most sales?
3. How has profit changed over time?
4. Who are the top customers by revenue?
5. Which shipping modes are fastest and most profitable?

## Files
- 'Raw dataset': https://www.kaggle.com/datasets/vivek468/superstore-dataset-final?resource=download
- `results_table.xlsx`: Contains outputs from SQL queries and  SQL code for analysis  
- `dashboard_screenshot.png`: Tableau dashboard preview
